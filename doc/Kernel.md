# Kernel

为一个目标字段定义其关联事物.

## 构造函数

| 参数 | 含义 |
| :---: | :---: |
| root | 目标对象 |
| alias | 目标字段属性名称（路径） |
| relations | 目标字段关联事物 |
| relations.dnstream | 关联的下游字段 |
| relations.resultIn | 赋值影响 |
| relations.upstream | 关联的上游字段 |
| relations.resultFrom | 取值函数 |
